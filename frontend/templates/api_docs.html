API Documentation

This is the API documentation for the Tattoo Studio System.

Authentication

All API endpoints require authentication. Use JWT tokens or session cookies.

Example Endpoints

  • GET /api/profile - Get user profile
  • POST /api/logout - Logout
  • GET /api/dashboard - Dashboard data
  • GET /api/health - Health check

Usage Examples

# Get user profile
curl -H "Authorization: Bearer <token>" http://localhost:5000/api/profile

# Health check
curl http://localhost:5000/api/health
